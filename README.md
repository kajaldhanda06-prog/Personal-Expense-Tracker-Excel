# Personal Expense Tracker (Excel)

An interactive Excel dashboard designed to track daily expenses, analyze spending habits, and manage monthly budgets using automation and visualization.

## 🚀 Key Features
* **Automated Summaries:** Uses Pivot Tables to summarize expenses by category and date.
* **Interactive Dashboard:** Includes **Slicers** for quick filtering by Month, Category, or Payment Method (e.g., UPI, Cash, Card).
* **Trend Analysis:** Visualizes spending patterns over time using Line and Bar charts.
* **Expense Breakdown:** A Pie Chart view to see which categories (Food, Rent, Travel, etc.) consume the most budget.

## 📊 How to Use
1.  **Download:** Clone this repository or download the `Personal Expense Tracker.xlsx` file.
2.  **Input Data:** Navigate to the 'Data' sheet and enter your daily transactions.
3.  **Refresh:** Go to the 'Dashboard' tab, right-click any chart, and select **Refresh** to update the visuals with your new data.
4.  **Filter:** Use the Slicers on the side to drill down into specific spending periods.

## 🛠️ Built With
* **Microsoft Excel:** Data management and formula logic.
* **Pivot Tables & Charts:** Dynamic data aggregation.
* **Slicers:** For a user-friendly, interactive interface.
* ## 📁 Data Structure
This project tracks expenses across the following main categories:

| Column Name | Description | Example Data |
| :--- | :--- | :--- |
| **Date** | The day the transaction occurred | 05-04-2026 |
| **Category** | The type of expense | Food, Rent, Travel, Bills |
| **Amount** | The total cost in currency | 500 |
| **Payment Mode** | How the payment was made | UPI, Cash, Credit Card |
| **Notes** | Brief details about the spend | Lunch at Cafe |

> **Note:** The "Payment Mode" column is specifically used to filter **UPI transactions** for monthly digital spending analysis.
